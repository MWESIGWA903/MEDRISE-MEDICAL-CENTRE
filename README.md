# Medrise Medical Centre

Medrise Medical Centre is a healthcare management platform designed to help patients and administrators manage medical services efficiently.

## Features

- Patient management
- Appointment scheduling
- Admin dashboard
- Secure login system
- Medical records management

## Technology Stack

- Frontend: React
- Backend: Node.js
- Database: PostgreSQL
- Deployment: Render

## Project Structure

- Frontend application
- Backend API
- Database services

## Status

Project currently under active development.
